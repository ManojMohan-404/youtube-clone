A front-end clone of the YouTube homepage built with HTML and CSS, focusing on replicating the core layout and design of the video browsing interface.

## Features

-   **Navigation Bar**: A responsive top navigation bar that includes:
    -   YouTube logo and a menu icon.
    -   A search bar with search and voice search buttons, including tooltips.
    -   User-related icons for video upload, app options, notifications, and user profile.
-   **Sidebar**: A vertical sidebar for main navigation links such as Home, Explore, Subscriptions, Library, and Music.
-   **Video Grid**: A responsive grid layout that displays a collection of video cards.
-   **Video Cards**: Each video card consists of:
    -   A video thumbnail with the video duration displayed.
    -   The channel's profile picture.
    -   Video details including the title, channel name, and view statistics.

## Technologies Used

This project was built using the following technologies:

-   **HTML5**: For structuring the content and elements of the web page.
-   **CSS3**: For styling, layout, and creating a responsive design.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need a modern web browser like Google Chrome, Firefox, or Microsoft Edge.

### Installation

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/ManojMohan-404/youtube-clone.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd youtube-clone
    ```
3.  Open the `index.html` file in your web browser.
